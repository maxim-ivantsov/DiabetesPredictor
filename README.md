# 🩺 Diabetes Predictor

Десктопное приложение на Python для предсказания диабета с помощью машинного обучения. Загружайте датасеты, обучайте модель и получайте прогнозы с вероятностями прямо в интерфейсе.

## 📸 Возможности

- 📂 Загрузка данных из **Excel (.xlsx)** и **CSV** файлов
- 🤖 Обучение модели **Random Forest** прямо в приложении
- 🔮 Прогнозирование по введённым данным пациента
- 📊 Визуализация данных (scatter, histogram, boxplot)
- 📋 История всех прогнозов
- 📄 Экспорт результата в **PDF**
- 🌙 Тёмная тема

## 🏗️ Архитектура

```
├── main.py                          # Точка входа
├── requirements.txt                 # Зависимости
└── diabetes_prediction_report.pdf   # Пример экспорта (генерируется)
```

### Вкладки приложения

| Вкладка | Описание |
|---|---|
| Данные | Просмотр, добавление, редактирование и удаление строк датасета |
| Прогнозирование | Ввод данных пациента и получение прогноза |
| Визуализация | Построение графиков по признакам |
| История | Лог всех выполненных прогнозов |

## 🤖 Модель

- **Алгоритм:** Random Forest Classifier (100 деревьев)
- **Предобработка:** StandardScaler
- **Разбивка:** 80% train / 20% test
- **Целевая переменная:** `Diabetes_012` (0 — нет диабета, 1 — предиабет, 2 — диабет)

### Формат входного файла

Файл должен содержать столбец `Diabetes_012` и любые признаки пациента. Пример совместимого датасета: [Diabetes Health Indicators Dataset (Kaggle)](https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset)

## 📋 Как пользоваться

1. **Загрузить данные** — Файл → Открыть Excel/CSV файл
2. **Обучить модель** — Модель → Обучить модель (или кнопка на вкладке Данные)
3. **Сделать прогноз** — перейти на вкладку Прогнозирование, заполнить поля, нажать «Сделать прогноз»
4. **Экспортировать** — кнопка «Экспорт в PDF» сохраняет последний результат
